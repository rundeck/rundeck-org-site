Looking for enterprise features and professional support? Try Rundeck Enterprise.

Looking for the open source community version? Choose an install option below.

Self Contained Launcher Install

{{site.rd_war_name}}

Put it in a directory named ~/rundeck

Execute:

java -jar {{site.rd_war_name}}

Debian/Ubuntu Install

{{site.rd_deb_name}}

Execute:

apt-get install openjdk-8-jdk-headless
dpkg -i {{site.rd_deb_name}}

Or fetch from packages.rundeck.com.

Yum RPM Install

bash <(curl https://raw.githubusercontent.com/rundeck/packaging/main/scripts/rpm-setup.sh) rundeck
yum install java-1.8.0 rundeck
service rundeckd start

Upgrade:

yum upgrade rundeck rundeck-config

Or fetch from packages.rundeck.com.

Docker

Hosted with documentation at Docker Hub

docker pull rundeck/rundeck:{{site.rd_short_version}}
